A React application that allows users to search GitHub profiles using **username**, **location**, and **minimum repository count**.

---

## Tech Stack
- Vite + React
- Axios
- Tailwindcss
- React Router DOM
- Zustand (state management)
- GitHub REST API

---

## Features
- Search GitHub users by:
  - Username
  - Location
  - Minimum number of repositories
- Display user information:
  - Name
  - Location
  - Public repository count
- Direct link to the GitHub profile using a **View Profile** button
